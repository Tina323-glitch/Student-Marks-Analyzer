# Student Marks Analyzer

This is a C++ project that analyzes student marks in one or more subjects using arrays.

## Features
- Calculates total and average marks
- Finds highest, lowest, and second highest marks
- Checks if marks are sorted
- Counts students scoring above average
- Can be extended to multiple subjects

## How to Run
1. Clone this repository
2. Compile the code: `g++ marks_analyzer.cpp -o marks_analyzer`
3. Run the program: `./marks_analyzer`
